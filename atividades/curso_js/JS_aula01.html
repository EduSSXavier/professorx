Introdução ao JavaScript

A Linguagem da Web

📚 Teoria

JavaScript é uma linguagem de programação dinâmica e interpretada, criada em 1995 por Brendan Eich. É amplamente utilizada para:

  • Desenvolvimento Web Frontend: Interatividade em páginas web
  • Desenvolvimento Backend: Servidores com Node.js
  • Aplicações Mobile: React Native, Ionic
  • Aplicações Desktop: Electron

É uma linguagem orientada a objetos, funcional e fracamente tipada.

JavaScript é uma linguagem de programação amplamente utilizada para desenvolvimento web. É uma das três principais tecnologias da web, juntamente com HTML e CSS.

💡 Exemplo

// Primeiro programa em JavaScript
console.log("Olá, mundo!");

🎯 Exercício

Desafio: Abra o console do navegador (F12) e digite: alert("Meu primeiro JavaScript!")

Objetivo: Familiarizar-se com a execução de código JavaScript no navegador.

Como Adicionar JavaScript no Documento HTML?

📚 Teoria

Existem duas maneiras de adicionar JavaScript no HTML e fazê-las trabalhar juntas:

  • Dentro do próprio dicumento HTML
  • Itilizando um documento externo

💡 Exemplo 1

<!DOCTYPE html>
<html lang="en-US">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <script>JAVASCRIPT PODE SER COLOCADO AQUI</script>
        <title>Págia de Teste</title>
    </head>
    <body>
        <script>JAVASCRIPT PODE TAMBÉM SER COLOCADO AQUI</script>
    </body>
</html>

💡 Exemplo 2

<!DOCTYPE html>
<html lang="en-US">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <title>Págia de Teste</title>
    </head>
    <body>
        <script src="meuscript.js"></script>
    </body>
</html>

ALERT()

Comando para exibir mensagens de alerta

📚 Teoria

O comando alert() é uma função global que exibe uma caixa de diálogo modal com uma mensagem e um botão "OK".

  • Sintaxe: alert(mensagem)
  • Parâmetro: String com a mensagem a ser exibida
  • Comportamento: Bloqueia a execução até o usuário clicar "OK"
  • Uso: Avisos, notificações simples, debugging básico

💡 Exemplo

// Exemplo básico
alert("Bem-vindo ao sistema!");

// Com variável
var nome = "João";
alert("Olá, " + nome + "!");

// Com quebra de linha
alert("Primeira linha\nSegunda linha");

🎯 Exercício

Desafio: Crie três alertas em sequência:

  • 1º: "Iniciando o programa"
  • 2º: "Processando dados..."
  • 3º: "Programa finalizado!"

CONSOLE.LOG()

Comando para exibir mensagens no console do navegador

📚 Teoria

O console.log() é o método mais usado para debugging e exibição de informações no console do desenvolvedor.

  • Vantagens: Não interrompe o fluxo, aceita múltiplos parâmetros
  • Variações: console.error(), console.warn(), console.info()
  • Uso profissional: Debugging, monitoramento, logs de desenvolvimento
  • Acesso: F12 → Console (na maioria dos navegadores) ou Ctrl + Shift + J

💡 Exemplo

// Exemplo básico
console.log("Mensagem no console");

// Múltiplos valores
console.log("Nome:", "Maria", "Idade:", 25);

// Com variável
var nome = "Maria";
console.log("Olá, " + nome + "!");

// Diferentes tipos de log
console.error("Erro encontrado!");
console.warn("Atenção: operação demorada");
console.info("Informação: dados carregados");

🎯 Exercício

Desafio: Use console.log para exibir:

  • Seu nome e idade
  • O resultado de 10 + 15
  • Uma mensagem de erro simulada
  • Suas 3 cores favoritas

PROMPT()

Entrada de dados do usuário

📚 Teoria

O prompt() exibe uma caixa de diálogo que solicita entrada do usuário.

  • Sintaxe: prompt(mensagem, valorPadrao)
  • Retorno: String com o valor digitado ou null se cancelado
  • Conversão: Sempre retorna string, necessário converter para números
  • Limitação: Interface básica, não recomendado para aplicações modernas

💡 Exemplo

// Exemplo básico
var nome = prompt("Digite seu nome:");
console.log("Olá, " + nome);

// Com valor padrão
var idade = prompt("Digite sua idade:", "18");

// Convertendo para número
var numero = parseInt(prompt("Digite um número:"));
var resultado = numero * 2;
console.log("O dobro é:", resultado);

🎯 Exercício

Desafio: Crie um programa que:

  • Peça o nome do usuário
  • Peça a idade (converta para número)
  • Calcule o ano de nascimento (2024 - idade)
  • Exiba: "Olá [nome], você nasceu em [ano]"

DOCUMENT.WRITELN()

Escrita direta no documento HTML

📚 Teoria

O document.writeln() escreve conteúdo diretamente no documento HTML seguido de uma quebra de linha.

  • Uso histórico: Muito usado no início do JavaScript
  • Limitações: Sobrescreve conteúdo se usado após carregamento
  • Alternativa moderna (que veremos futuramente): innerHTML, createElement, appendChild
  • Diferença: writeln() adiciona quebra de linha, write() não

💡 Exemplo

// Exemplo básico
document.writeln("<h1>Título da Página</h1>");
document.writeln("<p>Este parágrafo foi criado com JavaScript</p>");

// Com variáveis
var data = new Date();
document.writeln("<p>Data atual: " + data + "</p>");

// Criando lista
document.writeln("<ul>");
document.writeln("<li>Laranja (primeiro item)</li>");
document.writeln("<li>Maçã (segundo item)</li>");
document.writeln("<li>Banana (terceiro item)</li>");
document.writeln("</ul>");

// Alternativa moderna (recomendada)
document.getElementById("conteudo").innerHTML = "<p>Conteúdo dinâmico</p>";

🎯 Exercício

Desafio: Use document.writeln() para criar:

  • Um título "Minha Página Dinâmica"

  • Uma lista com seus 3 hobbies favoritos
  • Um link para seu site favorito

Variáveis

Armazenamento de dados na memória

📚 Teoria

Variáveis são contêineres para armazenar dados. JavaScript possui três formas de declará-las:

  • var: Escopo global/função, permite redeclaração
  • let: Escopo de bloco, não permite redeclaração no mesmo escopo
  • const: Escopo de bloco, valor constante (não pode ser reatribuído)

Regras de nomenclatura: Iniciar com letra, $, ou _; sensível a maiúsculas e minúsculas.

💡 Exemplo

// Declaração com var
var nome = "Maria";
var idade = 25;

// Declaração com let
let cidade = "São Paulo";
let salario = 5000.50;

// Declaração com const
const PI = 3.14159;
const EMPRESA = "Tech Solutions";

// Múltiplas declarações
var x = 10, y = 20, z = 30;

// Variável sem valor inicial
var resultado;
console.log(resultado); // undefined

// Reatribuição
nome = "João"; // OK
cidade = "Rio de Janeiro"; // OK
// PI = 3.14; // ERRO! const não pode ser reatribuído

🎯 Exercício

Desafio: Declare variáveis para:

  • Seu nome completo (const)
  • Sua idade atual (let)
  • Sua cidade (var)
  • Se você é estudante (let, boolean)
  • Exiba todas no console ou no documento HTML

Tipos de Dados

Diferentes categorias de informação

📚 Teoria

JavaScript é dinamicamente tipado. Os tipos principais são:

  • Number: Números inteiros e decimais (1, 3.14, -5)
  • String: Texto entre aspas ("João", 'Maria', `Template`)
  • Boolean: Verdadeiro ou falso (true, false)
  • Undefined: Variável declarada sem valor
  • Null: Valor nulo intencionalmente definido
  • Object: Coleção de propriedades {nome: "João"}

💡 Exemplo

// Number
let idade = 25;
let altura = 1.75;
let temperatura = -10;

// String
let nome = "Ana";
let sobrenome = 'Silva';
let frase = `Olá, ${nome}!`; // Template literal

// Boolean
let ativo = true;
let concluido = false;

// Undefined e Null
let indefinido;
let nulo = null;

// Object
let pessoa = {
    nome: "Carlos",
    idade: 30,
    ativo: true
};

// Verificando tipos
console.log(typeof idade);     // "number"
console.log(typeof nome);      // "string"
console.log(typeof ativo);     // "boolean"
console.log(typeof indefinido); // "undefined"
console.log(typeof pessoa);    // "object"

🎯 Exercício

Desafio: Crie variáveis de cada tipo e use typeof para verificar:

  • Um número decimal (sua altura)
  • Uma string (seu filme favorito)
  • Um boolean (se você gosta de programar)
  • Um objeto (dados de um livro: título, autor, páginas)

Operadores Aritméticos

Realizando cálculos matemáticos

📚 Teoria

JavaScript oferece operadores para operações matemáticas:

  • + (Adição): Soma valores ou concatena strings
  • - (Subtração): Subtrai o segundo valor do primeiro
  • * (Multiplicação): Multiplica dois valores
  • / (Divisão): Divide o primeiro pelo segundo valor
  • % (Módulo): Resto da divisão inteira
  • ** (Exponenciação): Eleva à potência (ES2016+)

💡 Exemplo

// Operações básicas
let a = 10, b = 3;

console.log("Adição:", a + b);        // 13
console.log("Subtração:", a - b);     // 7
console.log("Multiplicação:", a * b); // 30
console.log("Divisão:", a / b);       // 3.333...
console.log("Módulo:", a % b);        // 1
console.log("Potência:", a ** b);     // 1000

// Precedência dos operadores
let resultado = 2 + 3 * 4;  // 14 (não 20)
let comParenteses = (2 + 3) * 4; // 20

// Cuidado com strings
console.log("10" + 5);    // "105" (concatenação)
console.log("10" - 5);    // 5 (conversão automática)
console.log("10" * 2);    // 20 (conversão automática)

// Operações com decimais
let preco = 19.99;
let desconto = 0.1; // 10%
let valorFinal = preco - (preco * desconto);
console.log("Preço final:", valorFinal.toFixed(2));

  • Peça peso em kg

Operadores de Atribuição

Atribuindo e modificando valores

📚 Teoria

Operadores que atribuem valores às variáveis, alguns também fazem operações:

  • = (Atribuição simples): Atribui valor à variável
  • += (Adição e atribuição): x += 5 → x = x + 5
  • -= (Subtração e atribuição): x -= 3 → x = x - 3
  • *= (Multiplicação e atribuição): x *= 2 → x = x * 2
  • /= (Divisão e atribuição): x /= 4 → x = x / 4
  • %= (Módulo e atribuição): x %= 5 → x = x % 5
  • **= (Exponenciação e atribuição): x **= 2 → x = x ** 2

💡 Exemplo

// Atribuição simples
let contador = 0;
let nome = "JavaScript";

// Operadores de atribuição compostos
contador += 1;    // contador = contador + 1 (agora é 1)
contador += 5;    // contador = contador + 5 (agora é 6)
contador -= 2;    // contador = contador - 2 (agora é 4)
contador *= 3;    // contador = contador * 3 (agora é 12)
contador /= 4;    // contador = contador / 4 (agora é 3)
contador %= 2;    // contador = contador % 2 (agora é 1)

// Com strings
let mensagem = "Olá";
mensagem += " mundo";  // "Olá mundo"
mensagem += "!";       // "Olá mundo!"

// Incremento e decremento
let x = 10;
x++;  // x = x + 1 (pós-incremento)
++x;  // x = x + 1 (pré-incremento)
x--;  // x = x - 1 (pós-decremento)
--x;  // x = x - 1 (pré-decremento)

// Exemplo prático: sistema de pontos
let pontos = 100;
pontos += 50;    // Ganhou pontos
pontos -= 20;    // Perdeu pontos
pontos *= 1.5;   // Bônus de 50%
console.log("Pontos finais:", pontos);
